How it works

Taurus è un'interfaccia universale di pagamento per trasferimenti bancari UE / SEPA.

Secret Code

Inizia l'operazione con il codice segreto fornito dalla tua banca, che sarà anche la tua password di accesso a Taurus. Per effettuare i trasferimenti bancari dovrai sempre avere a portata di mano il tuo smartphone.

Operations

Le operazioni di pagamento avvengono attraverso un protocollo sicuro. Ogni operazione è eseguita grazie a un codice di sicurezza che può essere utilizzato una sola volta e che ti verrà inviato sul tuo smartphone.

Commit

Le operazioni vengono sincronizzate con tua banca. L'operazione può essere modificata o eliminata entro 8 ore dall'invio dell'ordine. Il tuo istituto banbario eseguirà l'ordine solamente il giorno seguente dopo aver effettuato i dovuti controlli.
